## How to Add Images

- **Hero image:** `heroImage: "/images/markers/kalambo-falls.jpg"` in narrative. Place assets in `public/images/markers/`.
- **Inline images:** Use `blocks` with `{ type: "image", src: "/path.jpg", caption: "Optional" }`.
- **Quote block:** `{ type: "quote", content: "...", attribution: "Source" }`.
